# AdvancedMission
내일배움캠프 안드로이드 숙련주차 개인과제입니다.

### 주요 기능
- 리스트에 보이는 상품목록을 선택하면 상세페이지로 이동합니다.
- 각 상품에 좋아요(하트)를 할 수 있습니다.
- 종모양Icon을 클릭시 Notification이 알림음과함께 상단에 상태창에 표시됩니다.
- 상단 스피너에서 원하는 동을 선택하실 수 있습니다.
- 뒤로가기 클릭시 종료Dialog가 표시되며 확인을 클릭시 앱이 종료됩니다.
